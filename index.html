What is NetRadar AI?

NetRadar AI is a session-based network intrusion detection system powered by machine learning. It combines real-time flow analysis with intelligent threat classification to protect modern networks.

Real-Time Packet Analysis

Live network traffic inspection using Scapy with low latency for immediate threat detection.

Flow-Based Metadata Extraction

Session-based architecture captures 11 network flow features — metadata only, no raw packet contents.

ML Threat Classification

A Decision Tree model trained on the CIC-IDS-2017 dataset classifies each flow for accurate attack prediction.

Lightweight Desktop App

Runs locally as a desktop application — no data sent to the cloud, privacy-friendly by design.
